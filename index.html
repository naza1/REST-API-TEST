Welcome Padawan

Torombolo is a REST API testing framework built on .NET that makes testing API endpoints easy, fast, and fun. Read below for a quick overview, or check out the API documentation.

  • Transform to Run Test on different environment
  • Data base connection support
  • Support HTTP 1.1 Standars POST, GET, DELETE, PUT
  • Collection of data mockup
